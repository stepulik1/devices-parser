# World Devices Parser

Асинхронный масштабируемый веб-парсер для извлечения древовидных технических характеристик товаров с e-commerce платформ. 

Проект спроектирован с упором на отказоустойчивость, обход антифрод-систем и бережное отношение к ресурсам сервера.



## Технологический стек

*   **Python 3.10+** (асинхронный стек `async/await`).
*   **Playwright** (асинхронный браузер для рендеринга динамического контента).
*   **Playwright Stealth** (маскировка браузера от антибот-систем вроде Cloudflare).
*   **lxml** — используется для парсинга HTML. Обработка DOM-дерева вынесена из процесса браузера в движок `lxml`, что существенно снижает нагрузку на CPU и оперативную память.
*   **aiosqlite** — асинхронный менеджер базы данных SQLite для параллельной работы с диском без блокировки основного потока.



## Архитектура и ключевые фичи

Проект разделен на изолированные модули:

1.  **`src/browser.py` (Управление браузером):**
    *   Генерирует реалистичные User-Agent, локаль и размеры экрана.
    *   Интегрирует `Stealth` для маскировки отпечатков.
    *   Отслеживает статус-коды 403, 429, 503 для своевременного обнаружения блокировок.
    *   Реализует механизм повторных попыток загрузки с экспоненциальным временем ожидания.
2.  **`src/database.py` (Хранение данных):**
    *   Нормализованная реляционная база данных (таблицы: Категории $\rightarrow$ Товары $\rightarrow$ Характеристики).
    *   Запись через асинхронный `UPSERT` для предотвращения дублирования данных при повторном запуске.
    *   Поддержка каскадного удаления спецификаций при обновлении товара.
3.  **`src/parser.py` (Логика извлечения и оркестрация):**
    *   **Обход категорий:** собирает все доступные категории и планомерно обходит их.
    *   **Сбор товаров:** парсит первые 5 товаров в каждой категории (лимит легко настраивается в коде).
    *   **Умное чтение карточки:** парсер спроектирован так, чтобы не падать при отсутствии у товара технических характеристик (например, на страницах простых чехлов и аксессуаров). Такие товары все равно сохраняются в базу, а парсинг характеристик корректно пропускается.
    *   **Рандомизация задержек:** защищает от бана за счет случайных пауз (`asyncio.sleep`) между запросами.
    *   **Защита от дублей:** отслеживает уже посещенные ссылки в текущей сессии через `visited_urls`.
